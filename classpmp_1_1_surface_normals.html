Detailed Description

A class for computing surface normals.

This class provides a set of static functions for computing surface normal information, either

  • per vertex: computeVertexNormal()
  • per face: computeFaceNormal()
  • per corner: computeCornerNormal()

The convenience functions computeVertexNormals() and computeFaceNormals() compute the normals for the whole mesh and add a corresponding vertex or face property.
